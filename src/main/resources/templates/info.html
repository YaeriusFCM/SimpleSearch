Application description

The application searches through a database with 500 top rated movies by IMDb.

Whenever you enter a character in the search bar, the application will run an AJAX query to find all matching actors' names in the database. Those names will be sorted by popularity (the biggest number of movies starring that actor), then truncated only to 5 names and those names will be displayed in alphabetical order. You can choose from the suggested names but you do not have to. You may just search for partial name if you like.

When you press 'Enter' or click the 'Search' button, the application will look for movies starring actors whose names match to the given query. Movies will be displayed in a table, ordered by rank (highest IMDb rating), paginated to fit maximum of 10 entries per page.

Technology stack used in this project: Spring Boot, Groovy, MongoDB, JSON, Thymeleaf, HTML, Bootstrap, CSS, Font Awesome, jQuery.
